## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client → upstream JSON response | Strict-decoding rejects responses with unknown fields; lenient (default) accepts them. The boundary is unchanged from Phase 2 — only the decoder gate is new. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-02 | Tampering | upstream JSON injecting unknown fields to confuse decoder | mitigate | `WithStrictDecoding(true)` surfaces schema drift loudly via `decoder.DisallowUnknownFields()` (D-92). Default-off matches Pitfall JSON-1 — upstream adds fields routinely, and silent rejection would break consumers. |
