| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-3-InputVal-Country | Tampering / DoS | subdivisions.go (validator call) | mitigate | validateCountry runs before any HTTP; rejects non-ASCII / Unicode-fold tricks (W-01 hardening). |
| T-3-InputVal-Lang | Tampering | subdivisions.go (validator call) | mitigate | validateLanguage runs when LanguageIsoCode is non-empty. |
| T-3-DoS-EmptyQueryParam | DoS (self-inflicted) | subdivisions.go query builder | mitigate | Empty-string guard for LanguageIsoCode (D-55). |

| T-3-InfoDisc-APIErrorBody | InfoDisc | doJSONGet (inherited) | mitigate | 4 KiB APIError.Body cap inherited. |
