<objective>
Land the demo CLI core: `cmd/ohcli/` with subcommand dispatch (`public`, `school`, `countries`, `version`, `help`), three output renderers (text via `text/tabwriter`, JSON via `encoding/json`, CSV via `encoding/csv` per D-03), exit-code policy (0/1/2 per D-06), stderr error prefix (`ohcli: ` per D-05), empty-result handling (stderr message + exit 0 per D-07), and the OPENHOLIDAYS_BASE_URL test seam Plan 02 will rely on.

Purpose: Dogfood the public library surface (CLI-03) and demonstrate the value proposition for v0.1.0 — `ohcli public PL 2025` prints all 14 Polish public holidays; `ohcli school PL 2025 --region PL-SL` filters to the Śląskie cohort. Stdlib-only (no `cobra`, no `urfave/cli`) per PROJECT.md zero-dep rule.

Output:
- `cmd/ohcli/main.go` — entrypoint, `run()`, dispatch, usage text.
- `cmd/ohcli/{public,school,countries}.go` — per-subcommand handlers.
- `cmd/ohcli/format.go` — render(w, hs, lang, format) dispatching to text/json/csv.
- `cmd/ohcli/version.go` — `ohcli version` resolution (debug.ReadBuildInfo first, fallback to library `Version`).
